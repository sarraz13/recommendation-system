# Recommendation System

## Overview
This project implements a **recommendation system** in Python to provide personalized suggestions to users based on their past interactions and product features.  
It combines **collaborative filtering** with **content-based filtering** using **cosine similarity and other methods** on product descriptions.

---

## Features
- Generate recommendations for users based on **similarity with other users** (collaborative filtering).  
- Use **product description analysis** (NLP) to improve recommendations.  
- Store and retrieve user and product data from a **PostgreSQL database**.  
- Output top N recommended items for each user.  

---

## Tech Stack
- **Languages & Libraries:** Python, NLTK, sklearn,flask
-  **Framework:** Flask
- **Database:** PostgreSQL  
- **Other Tools:** psycopg2 for database interaction, html/css for interfaces
